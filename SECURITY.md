# Security

This repository is a static, sanitized showcase and does not accept secrets,
credentials, private logs, device captures, or production configuration.

Please do not open public issues containing sensitive information. Report a
security concern privately to `legacycreator@protonmail.com` with the subject
`Aurelia showcase security`.

The public demo must remain read-only. It must not expose live device control,
private backend endpoints, authentication material, financial actions, or
sensitive user data.
