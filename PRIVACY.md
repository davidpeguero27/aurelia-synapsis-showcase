# Privacy

The Aurelia showcase is a static informational website. It does not request
account credentials, payment information, private codes, or access to a
visitor's device.

The showcase source contains no private Aurelia logs, user conversations,
device diagnostics, API keys, or operational secrets.

The hosting provider and GitHub Pages may process standard web-server metadata
under their own policies. The showcase itself does not add analytics,
advertising trackers, cookies, or form submissions.
